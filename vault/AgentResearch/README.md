# AgentResearch

Area owned by `src/agent_researcher`.

The autonomous research agent may write only here in the Obsidian vault. It
reads the broader vault for context, but all generated hypotheses, experiment
logs, and distilled learnings stay under this folder.

## Structure

| Folder | Index (MOC) | Purpose |
|---|---|---|
| `hypotheses/` | [[AgentResearch/hypotheses/_index\|hypotheses]] | Pre-test hypotheses generated by OpenCode. |
| `logs/` | [[AgentResearch/logs/_index\|logs]] | Full experiment records with research and holdout results. |
| `learnings/` | [[AgentResearch/learnings/_index\|learnings]] | Short reusable lessons fed back into future prompts. |

Cada arquivo individual linka de volta para o index do seu folder e para
este README via secao `## Related` no final.

## Safety Contract

- The agent does not write to `data/**`.
- Holdout usage is tracked in `src/agent_researcher/state.json`.
- Strategies are persisted only after research and holdout validation pass.
- Rejected or decayed strategies move from `strategies/active/` to
  `strategies/rejected/` inside `src/agent_researcher/`.
